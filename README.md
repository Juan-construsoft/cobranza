# Law Office Debt Management

This project is a lightweight internal web application designed for a law office to manage the collection of overdue accounts. The application automates initial steps and follow-ups without delving into complex legal processes.

## Features

### Module 1: Case Reception and Management
- Create and manage digital files for each obligation.
- Input fields for case details including debtor information, creditor details, and initial comments.
- Upload and link multiple documents related to each case.
- View and filter a list of all cases.

### Module 2: Demand Generation
- Generate drafts of demands based on predefined templates and case data.
- Download generated demands as DOCX or TXT files.

### Module 3: Activity Tracking and Alerts
- Record the progress of each case with a history of activities.
- Set deadlines for next steps and receive visual alerts for approaching or overdue deadlines.
- Automatic email notifications to responsible attorneys regarding deadlines.

### Module 4: Simple Reporting
- Dashboard providing an overview of active cases and their statuses.
- Export case lists to CSV or Excel format.

## Technology Stack
- React for building the user interface.
- TypeScript for type safety and better development experience.
- A simple database structure managed by the no-code platform.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd law-office-debt-management
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```

## Usage
- Access the application through your web browser.
- Follow the prompts to create cases, generate demands, track activities, and view reports.
